SCP Challenges

The source for all the challenges is publically available, but it's more fun to complete the challenge first!

To download the challenge, go to the releases page and then select the puzzle built for the OS relevant to you. On Linux and OSX, you may need to run chmod +x` on the binary before running it.
You are welcome to build the challenges yourself if you don't trust random code.
The apps are local servers, which will log which port they are listening on.

Available challenges

Puzzle 1: A CLI challenge. Can you solve the riddles that the endpoint gives you?

Puzzle 2: An HTML challenge! Can you find and submit all the secret codes?

Puzzle 3: A programming challenge! Retrieve problems from an endpoint, and solve them all in the time limit!

Puzzle 4: A programming challenge! See if you can get past the rate-limit on the server.

Puzzle 5: A local CLI challenge, can you figure out how to use the command line tool?
